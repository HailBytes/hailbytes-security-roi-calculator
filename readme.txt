== Description ==

The **HailBytes Security Training ROI Calculator** is a comprehensive WordPress plugin that helps organizations quantify the return on investment for security awareness training programs. Build compelling business cases with detailed cost-benefit analysis, industry benchmarking, and 5-year projections.

= Key Features =

* **Multi-Step Assessment Wizard** - Guided interface for comprehensive ROI analysis
* **Industry-Specific Benchmarks** - Benchmarks for 8 major industry sectors
* **Comprehensive Cost Modeling** - Detailed breakdown of training investments
* **Visual ROI Projections** - Interactive charts showing 5-year returns
* **Risk Assessment Engine** - Calculates current security posture and vulnerabilities
* **Training Effectiveness Modeling** - Based on proven methodologies and research
* **Detailed Recommendations** - Personalized next steps and implementation guidance
* **Export Capabilities** - Download results as CSV or share with stakeholders
* **Mobile Responsive Design** - Works perfectly on all devices
* **Professional Reporting** - Generate executive-ready business case documents

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins → Add New
3. Search for "HailBytes Security Training ROI Calculator"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Navigate to Plugins → Add New → Upload Plugin
4. Choose the downloaded ZIP file and click "Install Now"
5. Activate the plugin

= After Installation =

1. Add the shortcode `[security_training_roi_calculator]` to any page or post
2. Customize the calculator settings in Settings → ROI Calculator
3. Start calculating ROI for your security training programs!

== Usage ==

= Basic Usage =

Add the shortcode to any page or post:
`[security_training_roi_calculator]`

= Shortcode Parameters =

* `title` - Custom calculator title
* `show_hailbytes_branding` - Show/hide HailBytes promotion (true/false)

= Examples =

`[security_training_roi_calculator title="Security Training ROI Analysis"]`

`[security_training_roi_calculator show_hailbytes_branding="false"]`

= Widget Support =

The calculator can also be added to widget areas using the shortcode widget.

== Frequently Asked Questions ==

= How accurate are the ROI calculations? =

The calculator uses industry-standard benchmarks, peer-reviewed research, and proven methodologies for calculating security training ROI. Results are based on real-world data from cybersecurity studies and training effectiveness research.

= Can I customize the industry benchmarks? =

The current version uses research-based industry benchmarks. Customization options are available through our professional services or in premium versions.

= Does the plugin collect any data? =

The plugin optionally saves calculation results to your WordPress database for administrative review. No personal data is transmitted to external servers. You can disable data collection in the plugin settings.

= Can I export the results? =

Yes! You can export results as CSV files and share them with stakeholders. PDF export functionality is available in the premium version.

= Is the calculator mobile-friendly? =

Absolutely! The calculator is fully responsive and works perfectly on smartphones, tablets, and desktop computers.

= Can I use this for client presentations? =

Yes! The calculator generates professional, executive-ready reports perfect for client presentations and board meetings.

= What training costs are included? =

The calculator includes platform licensing, content development, staff time, external services, employee training time, tools, and ongoing operational costs.

= How are security benefits calculated? =

Benefits are calculated based on prevented incidents, reduced phishing success rates, compliance savings, insurance reductions, and productivity improvements using industry research and benchmarks.

= Can I customize the calculator appearance? =

The calculator uses standard WordPress theming and can be customized with CSS. Premium customization services are available.

Yes! We provide support through the WordPress.org forums and comprehensive documentation on our website.
